## 📖 About

Binance Trade Bot 2026 is a standalone Windows/Mac automation tool designed for executing automated trading strategies on the Binance cryptocurrency exchange. It provides a user-friendly interface for setting up and running algorithmic trades, market monitoring, and portfolio management without requiring manual intervention.

## ⚙️ Requirements

- Windows 10/11 (64-bit) or macOS 11+ (Intel/Apple Silicon)
- 4 GB RAM minimum (8 GB recommended)
- 200 MB  disk space
- Stable internet connection
- Binance account with API  (read and trade permissions)
- .NET Framework 4.8 (Windows) or compatible runtime (macOS)

- **Multi-Currency Support** 💱 — Trade hundreds of cryptocurrency pairs simultaneously
- **Risk Management** 🛡️ — Stop-loss, take-profit, and trailing stop features
- **Alert System** 🔔 — Email and desktop notifications for  events

Edit the `config.json` file generated after first launch to customize API , trading pairs, and strategy parameters:

{
  "exchange": "binance",
